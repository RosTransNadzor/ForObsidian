
- При создании экземпляра класса мы получаем ссылку на него

```cs
// 1 пример
Some some1 = new(){Value = 5};
Some some2 = some1;

some1 = new(){Value = 8};
// в итоге some1 и some2 - указывают на разные объекты

// 2 пример
Some some1 = new(){Value = 5};
Some some2 = some1 

// изменяем именно то куда ссылается some1, без ref пример бы такой же как 1 и на some1 изменения ссыки бы не отобразились
ref Some some1Ref = ref some1;
some1Ref = new(){Value = 8}

// в итоге some1 и some2 - указывают на разные объекты
// ref - полезен в методах где нет доступа к оригинальной ссылке
```
