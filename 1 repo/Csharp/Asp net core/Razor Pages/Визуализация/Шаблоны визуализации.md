[[Визуализация]]
- С# код в html
```cs
<h1>@DateTime.Now.Year</h1>
```
- Выражения c#
```cs
<p>@(1+2)</p>
```
- Условия
```cs
@if(1==2)
{
	<h1>False</h1>
}
else{
	<h1>Its true</h1>
}
```
- Циклы
```cs
@for(var i=0; i< tasks.Count; i++)
{ 
	var task = tasks[i];
	// динамическая выдача html
	<li>@i – @task<li>
}
```
