[[Конфигурация]]

- Подключение
```cs
// создание дефолтного сборщика 
// и передача в appConfiguration своей функции
Host.CreateDefaultBuilder()
	.ConfigureAppConfiguration(AddAppConfig);
```
```cs
// своя функция
static void AddAppConfig(HostBuilderContext context, IConfigurationBuilder config)
{
    // очищение дефолтных настроек
	config.Sources.Clear();
	// добавление чтения JSON файла и пропуск его при отсутствии(optional:true)
	config.AddJsonFile("appsettings.json", optional: true);
		  // этот файл загружается позже поэтому все значения в словаре у которых
		  // ключи совпадают будут перезаписаны 
	      .AddJsonFile("some.json",optional:true)
}
```
- Перезагрузка файлов
```cs
//reloadOnChange - автоматически перезагрузить при его изменении
// без остановки приложения
config.AddJsonFile( "appsettings.json", optional: true reloadOnChange: true);
```
- Получение данных из конфигурации
[[Теория (конфигурация)]]
```cs
// из примера заметки Теория(конфигурация)
// получение из словаря конфигурации
var zoomLevel = builder.Configuration["MapSettings:DefaultZoomLevel"];
```
