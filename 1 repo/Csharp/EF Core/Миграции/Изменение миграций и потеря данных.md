[[Миграции]]

- Иногда возникает необходимость изменить миграции
- К примеру во избежания потери данных 

- Нам нужно изменить имя столбца а миграция удаляет столбец и создает новый
```cs
migrationBuilder.AddColumn( 
	name: "UserId", 
	table: "Orders", 
	type: "uniqueidentifier", 
	nullable: false, 
	defaultValue:  new Guid("00000000-0000-0000-0000-000000000000")
);
migrationBuilder.DropColumn(name: "CustomerId",table: "Orders");
```
- Можно просто переименовать столбец
```cs
migrationBuilder.RenameColumn( name: "CustomerId", table: "Orders", newName: "UserId");
```
- Иногда для избежания потери данных можно использовать sql
```cs
// sql вместо миграции кодом
migrationBuilder.Sql("ALTER TABLE…")
```
