[[Многие ко многим]]

- Для этого необходимо определить 2 класса без связующей таблицы
```cs
public class Book{
    // ...
	public ICollection<Tag> Tags { get; set; }
}
public class Tag{
	// ... 
	public ICollection<Book> Books { get; set; }
}
```
- получение нав свойств см. здесь [[Загрузка связей]]
