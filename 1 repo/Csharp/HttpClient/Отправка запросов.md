[[HTTPClient]]
- Get Запрос
```cs
static HttpClient _client = new();
//...
_client.BaseAddress = new Uri("https://jsonplaceholder.typicode.com/");
_client.Timeout = TimeSpan.FromSeconds(5);
//...
using var response = await _client.GetAsync("/posts/1");
// вместо проверки можно вызывать исключение при плохом запросе с помощью
// response.EnsureSuccessStatusCode()
if (response.IsSuccessStatusCode)
{
// использование NewtonJsonSoft
    string resString = await response.Content.ReadAsStringAsync()
	Post? post = JsonConvert.DeserializeObject<Post>(resString);
	if(post != null)
	{
		Console.WriteLine(post);
	}
}
```
- Аналогично работает Delete
- Post - запрос
```cs
// содержимое запроса
using HttpResponseMessage response = await httpClient.PostAsJsonAsync(
	"todos", 
	new Todo(UserId: 9, Id: 99, Title: "Show extensions", Completed: false)
);

response.EnsureSuccessStatusCode()
// получение обратного результата
var jsonResponse = await response.Content.ReadAsStringAsync();
Console.WriteLine($"{jsonResponse}\n");
```
